# Desafio.Tech

# Sobre o projeto

Back-end da plataforma Desafio.tech.

Uma plataforma totalmente gratuita voltada a desafiar estudantes de programação a resolver problemas reais em 15 dias.

# Tecnologias utilizadas

- C#
- .Net
- Asp.Net
- Entity Framework
- Sql Server
